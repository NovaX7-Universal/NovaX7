📢 **Hello Editors!**
The Program is ready! It can download from anywhere! *(without DRM)*
It's called **NovaX7** — PC & Laptop only!

> Nova is a local media library packed with features:

🎵 **Library**
- Import local files & folders *(Audio + Video)*
- Automatic metadata & cover art extraction
- Favourites, Recently Played, Most Played views
- Playlist system with custom covers & descriptions
- NEW: Share playlists via code download

⬇️ **Downloads**
- Audio from YouTube, SoundCloud, Bandcamp, Vimeo & 1000+ sites
- Formats: MP3, M4A, FLAC, WAV, OGG — up to 320 kbps
- Video up to 4K — MP4, MKV, AVI, WEBM, MOV, GIF
- Built-in Media Browser *(YouTube & more, directly in the app)*

🔄 **Converter**
- Convert between any audio & video format
- Video → Audio, Audio → Video, Video → Video, Audio → Audio
- Compatible output *(H.264 + AAC)*

▶️ **Player**
- Full VLC-powered playback *(audio & video)*
- Fullscreen mode, Sleep Timer, Queue view
- Shuffle, Loop,
- 13 themes *(Midnight, Dracula, Nord, Catppuccin & more)*

⚙️ Installation (Windows)
First of all, install Python (open cmd)
`winget install Python.Python.3`
then:
1. Install VLC → videolan.org or:
`winget install VideoLAN.VLC`
2. Install Python dependencies:
`pip install PyQt6 python-vlc mutagen yt-dlp imageio-ffmpeg pypresence`
3. Optional (for built-in browser):
`pip install PyQt6-WebEngine`
Or press WINDOWS + R and type in `pip install -r requiremnts.txt`
(still need to download python and vlc for that)

5. Run in your PATH Terminal :
`python NovaX7_7_1_0.py`

⚙️ Installation (Mac)
You will need Homebrew for this one, so 
So Search for Spotlight  → CMD + Space → Type in Terminal
Type in this: `"/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
then: 
`brew install python --cask vlc && pip3 install PyQt6 python-vlc mutagen yt-dlp imageio-ffmpeg pypresence`
after that go into the explorer click on "new terminal in folder" (in the folder that nova is installed) and then type in:` python3 NovaX7_7_1_0.py`
